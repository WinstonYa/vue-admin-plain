## Getting started
```bash

# enter the project directory
cd vue-admin-plain

# install dependency
npm install

# develop
npm run serve
```

## Build
```bash
npm run build
```

## Advanced
```bash
# code format check
npm run lint

# code format check and auto fix
npm run lint --fix
```
